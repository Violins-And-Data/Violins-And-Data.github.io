Projects

Click below to see some of the projects I have done through my coursework.

  • Books Per Month Analysis

This project was done as a part of my Data Analytics in Business class and I got to utilize some of my training from Six Sigma and LEAN. I wanted to determine how to read more books during the year, so I did a root cause analysis with several methods, then a logistic regression to see what factors seemed to most impact my ability to finish a book.

Photo by Toa Heftiba on Unsplash

  • Dice Game Simulation

As a part of my Simulation coursework, I wrote code to simulate a dice game in which the number rolled determines the coins or points allotted to each player.

Photo by Brett Jordan on Unsplash

  • Random Variate Generation Library

This challenge with this project was to code a library that generates different distributions without using the built-in functions. My library includes Bernoulli, Binomial, and Geometric distributions as well as many others.

Photo by Carlos Muza on Unsplash

  • Recipe Optimization - AC:NH

The goal of this project was to create a program that takes input from the player and outputs the optimal mix of crafting recipes to maximize profit in the popular video game, Animal Crossing: New Horizons (AC:NH).
